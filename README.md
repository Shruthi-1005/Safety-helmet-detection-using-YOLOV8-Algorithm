# Safety-helmet-detection-using-YOLOV8-Algorithm

This project uses **YOLOv8 (You Only Look Once)** to detect whether a person is wearing a **safety helmet** or not.  
It can be used in construction sites, industries, factories, and safety monitoring systems to prevent workplace accidents.
